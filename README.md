# S3 Logs Web App con LocalStack

Aplicación web simple en Node.js + Express que permite guardar y listar logs en un bucket S3 simulado con LocalStack.  

## Requisitos previos

- **Docker Desktop** instalado y corriendo
- **LocalStack** corriendo
- **Node.js** instalado
- **AWS CLI** es opcional
- **npm install** para instalar dependencias del proyecto

# Para correr la app:
ejecturar node app.js

# La aplicacion corre en el puerto 3000 localhost
